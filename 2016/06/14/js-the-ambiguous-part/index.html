HTML中的 attribute 和 property

概述

attribute 和 property 是经常被混淆的两个概念。

区别

很多情况下，这二者是相互映射的，就是说 attribute 反射了同名的 property。但是另外的情况也不少。

名字不一致

最典型的是 className，为了回避 JavaScript 的保留字，跟 class attribute 对应的 property 是 className。

类型不一致

最典型的是 style，不接受字符串类型赋值。

语义不一致

如 a 元素的 href 属性。
